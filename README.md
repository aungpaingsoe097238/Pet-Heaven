
# Pet Adoption

Pet adoption website.


## Demo Link

https://pets-dun-nine.vercel.app/

## Tech Stack

**Client:** JavaScript, React, Redux, TailwindCSS




## Features

- SignIn, SignUp Form
- Pets List
- Pet Detail
- Adopt Form 
- Donate Form
- Responsive Device


## Running Tests

To run tests, run the following command


Installation Packages
```bash
  npm install 
```
Running With Development Server 
```bash
  npm run dev  
```
